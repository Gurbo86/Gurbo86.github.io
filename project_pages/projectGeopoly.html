ABOUT GEOPOLY

Geopoly is an economic simulation game based on real-world geolocation, where players buy, sell, and rent virtual properties tied to real places. Available in both mobile and NFT versions, Geopoly lets users play traditionally or through blockchain technology, offering real rewards and digital ownership. It blends casual gaming with the benefits of the Web3 ecosystem, delivering a unique experience for both gamers and crypto enthusiasts.

FEATURES

  • Geolocalized property purchace with homologes buildings in real-life.
  • Asyncronic interaction between player, creating turf coompetition.
  • Augmented Reality games that would provide player with more real-life interaction.
  • NFT and Cripto tokens with internal wallet providing players a total new experience.

Project Description

Geopoly was a super interesting project in which I have the chance to work as programmer, UI/UX designer and producer. It was challenging but I was able to learn a lot about other areas of development. The game when I started working on it needed a whole astetical and code revamp. Across one year I was able to create a new UI, introduce de AR minigames and improve base game mechanics.

Project Contributions

  • I developed the app fully in C# and Unity.
  • Worked side by side with the art department designing UI Flows.
  • I organized team pipelines and tasks through HackNPlan platform.
  • For almost 8 month I worked solo, then I managed a junior programmer.
  • Integrated Niantic SDK into the project to introduce AR minigames.
  • 80% of the development was made remote because of COVID.
