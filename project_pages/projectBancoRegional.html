ABOUT BANCO REGIONAL IMOBLIÁRIO

Banco Regional Imobiliáirio brings Brazil’s classic property-trading board game to mobile. Buy, sell and swap landmarks, build houses and hotels, collect rent and bankrupt rivals. Play lightning-fast matches online, offline or against a sharp AI, tweak rules to suit your table and track progress with samba-flavoured animations. Whether on the bus or at home, every roll can make you the next mogul.This game is conforms a bundle of games that belong to the "Jogos de Sempre" franchise, which includes other popular games like "War Strategy" or "A Vida e um Jogo".

FEATURES

  • Multiplayer asyncronic gameplay.
  • Play with friends or AI.
  • Cloud saves and global rankings.
  • Cross-platform between iOS, Android and web.
  • The game is crafted for all the family.

Project Description

Banco Regional Imobiliário started as a side assignment alongside my core duties, and—like anyone in a startup—I had to wear several hats. My chief responsibility was to add a full monetization layer, integrating in-app purchases into a Unity game already live on Web, Android and iOS. I designed and implemented the purchase flow, integrated store validation and analytics, and refactored legacy code so the new economy slotted in seamlessly. I also shipped dozens of bug fixes and quality-of-life tweaks that boosted stability and player retention.

Project Contributions
